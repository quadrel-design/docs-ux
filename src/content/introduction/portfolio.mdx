# Quadrel Product and Tools

Quadrel enables fast concept and implementation based on an existing UX architecture using a low-code approach, 
provides a broad functional scope through components and patterns, remains openly extensible via contribute repositories, 
and stays maintainable long-term by centralizing UI code and services.
